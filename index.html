Restoclub

A multi page fictional restaurant club website. Built using flexbox, the site is fully responsive and features a side bar, animations and a tip calculator that pops out in a modal.

The user can search for restaurants by area by typing in the search box on the homepage. When the user clicks the search button, the site connects to the Zomato API and displays the first twenty results

The tip calculator and its modal are built using JavaScript and allows the user to work out how much each person would need to pay for the tip.

The site features a number of animations, background videos and a fully functional contact form. The user can also add plans to a shopping card via the plans page. These items can then be cleared from the cart if the user wishes.

For mobile and tablet devices the sidebar snaps to the top of the screen underneath the user bar, which features a dropdown menu when the user picture is clicked.

Movie Collection

The application allows the user to search for a movie by the title. A request is then made to the Movie Database API and returns the first 20 results, which are then displayed in the UI.

The user can then choose to add a movie to their collection or wishlist. When the user clicks to add to one of the lists another request is made using the movies unique id and returns more detail about the chosen movie along with displaying it in the corresponding area of the UI.

When the user adds a movie to the collection or wishlist, the data is also persisted to local storage. The movies that are in local storage will be displayed in the UI upon the application being loaded.

Out-N-In

A website for a fictional fast food restaurant. Built using CSS grid, this is a modern fast food restaurant site.

The site features a Bootstrap nav bar and a landing section boasting an animated title. Along with the about section, contact form and footer the site also features a gallery section showcasing the power of CSS grid.

Upon clicking the order online button, a modal opens and the user is able to add and remove items to and from a cart, whilst the total price updates depending on whats in the cart.

The contact form is built from scratch and features JavaScript validation.

Colmar

The final project in the Codecademy: Build websites from scratch course, this is a single page scrolling website for the fictional Colmar academy.

The site has been built using Flexbox along with other display and positioning techniques. The end result is a simple, easy on the eye, information website containing a number of hover effects.

Color Game

A fun color guessing game built with HTML, CSS and JavaScript. The aim of the game is to guess which colored square matches the RGB value displayed at the top of the application.

The JavaScript generates six random colors everytime the application is loaded or restarted. The colors are displayed on the squares in the application and from these colors one of them is randomly picked as the one to guess.

The user then clicks on the square that they think is correct and each time they are wrong the square will fade away. Once the correct square is picked the header will change to the correct color along with all squares being displayed with that color.

The game features and easy and a hard mode where the user can choose between a three square game or a six square game.

Task List

A simple task list application built with HTML, CSS and JavaScript. The application allows a user to type in task items and display them in the application UI.

Depending on the users requirements they have the ability to remove individual tasks or clear all tasks from the application. The tasks are also persisted to local storage when the user adds them to the application. When the tasks are deleted they are removed from local storage at the same time.

The application features a filter input, where the user can type into the field and any task items containing that text will be displayed in the UI with the others hidden.
